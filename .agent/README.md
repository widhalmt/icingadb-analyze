# .agent/

This directory holds plans, checklists, decisions, and in-progress notes
produced by AI coding agents working on this project.

It is committed to the repository. It is project knowledge, not throwaway
state. A plan written here is useful to the next agent session, to other
agents, and to humans reviewing what was decided and why.

Files are Markdown. Plans use Markdown task list syntax with checkboxes.
Significant entries are date-stamped.

For the full mechanics — file format, resumability rules, the upstream
feedback loop — see `.roo/rules/00-agent-workflow.md`.
